# Device Maintenance Management App

This is a simple device maintenance management app for users to keep track of their devices' maintenance date. 

### Tech stack
- Nodejs 22.14.0
- Vuejs 3.5.17 with VueRouter

### The app contains 4 sections to manage: 
- Unit (The Unit or Department that the devices belong to)
- Device 
- Maintainer
- Maintenance Report. 

### Summary App Features
- User login with Google (OAuth2) using ```vue3-google-login```
- App supports CRUD, Filter and Search features (VueJS)
- API request is generated by ```Axios```.
- Client caching using ```@tanstack/vue-query```
- Server validation using ```zod```
- Message standardized with ```JSend```
- Database management and scripting using ```knex``` with connection by ```pg```
- Rate Limit using ```express-rate-limit```
- API documentation using ```swagger-ui-express```
